Praktikum der Finanzwirtschaft

Die hier implementierte Funktionalität umfasst die Auswertung einer Value at Risk sowie einer Conditional Value at Risk Analyse auf Basis des Monte Carlo Simulationsalgorithmus. Hierzu werden derzeitige echtdaten analysiert, undanschließend ausgewertet angezeigt. Die Parameter der Analyse sind hierbei selbst zu bestimmen.

Programmiert wurde das Programm in Python 3.6.0 sowie Html und Javascript. Als Framework wurde Bokeh eingesetzt, welches insbesondere die Darstellung der analyse sicherstellt.
